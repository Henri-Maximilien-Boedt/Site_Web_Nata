# Site Web NATA Bar

Site vitrine simple en HTML/CSS/JS pour **NATA Korean Food & Bar**.

## Lancer le site

1. Ouvre le dossier `Site_Web_Nata` dans VS Code.
2. Ouvre `maquette/index.html` dans ton navigateur.

Option recommandee dans VS Code:
- Installer l'extension **Live Server**
- Clic droit sur `maquette/index.html` > **Open with Live Server**

## Structure

- `maquette/index.html` : contenu du site
- `maquette/styles.css` : design responsive
- `maquette/script.js` : menu mobile + animations
- `maquette/media/img/` : images du logo et des plats
