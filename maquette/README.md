# Site Web NATA Bar

Site vitrine simple en HTML/CSS/JS pour **NATA Korean Food & Bar**.

## Lancer le site

1. Ouvre le dossier `Site_Web_Nata` dans VS Code.
2. Double-clique sur `index.html` pour l'ouvrir dans ton navigateur.

Option recommandÃ©e dans VS Code:
- Installer l'extension **Live Server**
- Clic droit sur `index.html` > **Open with Live Server**

## Structure

- `index.html` : contenu du site
- `styles.css` : design responsive
- `script.js` : menu mobile + animations
- `assets/images/` : images du logo et des plats
